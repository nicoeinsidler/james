# james

Utility CLI app to help with navigating to important folders and capture notes.

## Features

- create new single line note

  `james new +slug "this is a new note, perhaps during a meeting"`

  this will quickly append the line to the general file
- create multi line note

  `james new +slug`

  this will open the note in your configured text editor
- navigate and open current projects

  `james`
